## Run

- Node.js LTS download
  [https://nodejs.org](https://nodejs.org/ko/)

- npm download

```
$ npm install
```

- project run

```
$ npm run start
or
$ npx npm start
```

## Stack

- TypeScript
- react
- nextron.js
- firebase
- emotion
- recoil
